通信流程

云端 → MQTT → ESP32 → UART → ARM(模拟器)
                                    ↓
                              执行命令
                                    ↓
云端 ← MQTT ← ESP32 ← UART ← 返回结果
说明: ESP32收到MQTT命令后，直接转发原始JSON给ARM。ARM执行后通过UART返回结果，ESP32转发到MQTT。

命令列表

命令方向说明
set_control云端→设备设置远程控制参数
get_params云端→设备获取设备当前参数
set_params云端→设备设置设备参数
set_alarm云端→设备告警控制（触发/清除）
reset云端→设备设备重置

set_control - 设置控制参数

MQTT cs_inv/{sn}/cmd

请求

{
  "cmd": "set_control",
  "params": {
    "power_limit": 5000,
    "charge_enable": true,
    "discharge_enable": true,
    "max_charge_current": 60.0,
    "max_discharge_current": 120.0
  },
  "task_id": "cmd_1234567890_1234"
}

参数说明

字段类型必填说明
power_limitint否功率限制 (W)，默认6200
charge_enablebool否充电使能，默认true
discharge_enablebool否放电使能，默认true
max_charge_currentfloat否最大充电电流 (A)，默认60.0
max_discharge_currentfloat否最大放电电流 (A)，默认120.0

set_params - 设置设备参数

MQTT cs_inv/{sn}/cmd

请求

{
  "cmd": "set_params",
  "params": {
    "charge_control": true,
    "discharge_control": true,
    "power_limit": 5000,
    "max_charge_current": 60.0,
    "max_discharge_current": 120.0
  },
  "task_id": "cmd_1234567890_1234"
}

参数说明

字段类型说明
charge_controlbool充电控制开关
discharge_controlbool放电控制开关
power_limitint功率限制 (W)
max_charge_currentfloat最大充电电流 (A)
max_discharge_currentfloat最大放电电流 (A)

set_alarm - 告警控制

MQTT cs_inv/{sn}/cmd

触发告警

{
  "cmd": "set_alarm",
  "params": {
    "code": 1
  },
  "task_id": "cmd_1234567890_1234"
}

清除所有告警

{
  "cmd": "set_alarm",
  "params": {
    "code": 0
  },
  "task_id": "cmd_1234567890_1234"
}

告警代码

代码级别描述
1fault逆变器过温保护
2fault电池过压保护
3fault电池欠压保护
4fault输出过载保护
5fault直流母线过压
6warning逆变器温度过高
7warning电池SOC过低
8warningPV输入异常
9warning电芯压差过大

reset - 设备重置

MQTT cs_inv/{sn}/cmd

请求

{
  "cmd": "reset",
  "task_id": "cmd_1234567890_1234"
}

效果

  • SOC重置为75%
  • 清除所有告警
  • 状态恢复为"逆变运行"

执行结果格式

MQTT cs_inv/{sn}/cmd_result

{
  "task_id": "cmd_1234567890_1234",
  "cmd": "set_control",
  "success": true,
  "message": "控制参数已设置",
  "data": { ... },  // 可选，get_params返回
  "timestamp": 1782893097
}

控制字段说明

字段类型范围默认值说明
power_limitint0-62006200最大输出功率限制 (W)
charge_enablebooltrue/falsetrue是否允许充电
discharge_enablebooltrue/falsetrue是否允许放电
max_charge_currentfloat0-10060.0最大充电电流 (A)
max_discharge_currentfloat0-200120.0最大放电电流 (A)
注意: set_control 和 set_params 功能类似，但字段名略有不同。 set_control 使用 charge_enable/discharge_enable， set_params 使用 charge_control/discharge_control。 建议统一使用 set_params。
