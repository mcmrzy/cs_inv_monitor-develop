- **统一路由与权限控制**：在 `App.tsx` 中集中定义所有业务模块（如仪表盘、设备、告警等）的路由，并通过 `ProtectedRoute` 组件结合 `authStore` 实现全局身份校验与基于角色的页面重定向。
- **标准化 API 通信层**：通过 `services/api.ts` 封装 Axios 实例，利用请求/响应拦截器统一处理 JWT Token 注入、自动刷新及 401 异常跳转，各子模块 API（如 `deviceApi`, `alertApi`）均复用此基础实例。
- **全局状态与主题管理**：使用 Zustand 管理认证、时区及多语言状态，并在根组件通过 Ant Design 的 `ConfigProvider` 统一下发 UI 主题令牌与国际化 Locale，确保全应用视觉与文案的一致性。
